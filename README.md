# 📅 Cronograma de Estudos — Engenharia de Software

Página web desenvolvida com **HTML** e **CSS puro** para organizar e visualizar um cronograma semanal de estudos, criado para acompanhar a grade de disciplinas do curso de Engenharia de Software junto com trilhas complementares de desenvolvimento.

---

## 🖥️ Preview

> Abra o arquivo `index.html` no navegador ou acesse via [GitHub Pages](https://seuusuario.github.io/cronograma-estudos) para visualizar o projeto.

## ⏰ Rotina diária

| Horário | Atividade |
|---------|-----------|
| 05:00 | Acordar |
| 05:15 | Devocional & Oração |
| 06:00 | Academia |
| 07:30 | Casa em ordem |
| 08:00–09:30 | Estudos — manhã |
| 21:00–23:00 | Estudos — noite |

---

## ✨ Funcionalidades

- Grade semanal com blocos de manhã e noite
- Código de cores por disciplina
- Barras de progresso com carga horária estimada
- Rotina diária organizada em cards
- Layout responsivo para mobile
- Tema escuro com paleta de cores em variáveis CSS
